# Google Colab Notebooks

A collection of Jupyter notebooks for Google Colab and ComfyUI workflows for AI image generation and processing.

## Contents

- **notebooks/**: Jupyter notebooks for various AI tasks, including image upscaling, video processing, and ComfyUI setups.
- **comfyui-workflows/**: JSON workflows for ComfyUI, a powerful tool for generative AI image creation.
- **untested/**: Experimental or untested notebooks and workflows.

## Usage

1. For Colab notebooks: Upload to Google Colab or run locally with Jupyter.
2. For ComfyUI workflows: Import into ComfyUI for image generation.

## Requirements

- Python 3.x
- Jupyter Notebook or Google Colab
- ComfyUI for workflows
